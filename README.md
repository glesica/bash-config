# Bash Configuration

## Installation

The files can be installed using the included script:

    ./install.sh

## Stuff that gets installed

  * `.bashrc`
  * `.bash_aliases`
  * `.bash_profile`
  * `.bash_completion`
  * `.bash_private`
  * `.xsessionrc`
  * `.tmux.conf`
  * `.gitconfig`
  * `.git-prompt.sh`
  * `.git-completion.sh`
  * `.bin/*`
